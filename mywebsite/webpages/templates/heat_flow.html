heat flow

This program models heat flow in a 10 x 10 array. Value of each cell represents temperature of that cell. Temperature of the border (i.e. aij where i=0,9 and j=0-9 and aij where i=1-8 and j=0,9) is specified using the following form (all the cells in the border have the same temperature). Besides this, in the following form, a source (i.e. ij where i=1-8 and j=1-8) and its temperature is specified. Using these info, program iterates to calculate temperature of the other cells (i.e. cells other than the source and border). Here is the formula for the iterations:

aij = 1/5(aij + ai-1j + ai+1j + aij-1 + aij+1)

Iterations stop when ∑( aijnew - aijold ) 2 is sufficiently small (~0.5).
